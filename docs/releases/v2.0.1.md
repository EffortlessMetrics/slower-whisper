# slower-whisper v2.0.1

Released: 2026-02-16

This patch release formalizes the v2.0.1 feature set with no intended API-breaking changes.

## Highlights

- Post-processing orchestration via `PostProcessConfig` and `PostProcessor`.
- Topic segmentation support, including `TopicSegmenter` and `StreamingTopicSegmenter`.
- Turn-taking policy presets and end-of-turn confidence evaluation.

## Upgrade

```bash
pip install -U slower-whisper==2.0.1
# or your standard UV workflow
uv sync --extra full
```
